## Descripcion del Dataset
Dataset: Medical Cost Personal Dataset
Fuente: Kaggle
Variables: age, sex, bmi, children, smoker, region, charges

## Objetivo
Analizar que factores influyen en los gastos de seguros medicos usando Python, pandas, Plotly y DuckDB.

## Etapas del Proyecto
1. Carga y Exploracion de datos con pandas
2. EDA: valores nulos, duplicados y estadisticas
3. Visualizaciones con Plotly
4. 5 Consultas SQL usando DuckDB

## Tecnologias Usadas
Python, pandas, numpy, plotly, duckdb, Google Colab
